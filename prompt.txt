# Build a Modern Banking AI Agent (Full Stack)

You are an expert AI Engineer, Senior Software Architect, and UI/UX Designer.

Build a complete, production-quality Banking AI Agent using the latest technologies and best coding practices.

---

# Project Name

SmartBank AI Agent

---

# Goal

Create an AI-powered banking assistant that helps users understand banking products, answer questions, compare loans, calculate EMIs, summarize financial documents, and simulate banking services.

This is an educational/demo project only.

Never perform real banking transactions.

Generate the project incrementally.

Start by creating the folder structure and configuration files.

Then implement the backend.

Then AI agent architecture.

Then RAG.

Then authentication.

Then frontend.

Then dashboards.

Then testing.

Then Docker.

Ensure every generated file is complete and runnable.

Do not use deprecated libraries.

Use the latest stable versions available.

The application should run successfully after installing dependencies and configuring environment variables.

Produce clean, modular, production-ready code with clear explanations for each major component.
